Sprint 8 — UI/UX Polish (Mobile First)

OBIETTIVO

Non aggiungere funzionalità backend.
Voglio trasformare l'interfaccia attuale in una UI professionale, moderna e fluida, mantenendo tutta la logica esistente.

NON modificare:
- API
- WebSocket
- autenticazione
- database
- modelli
- endpoint

Modificare SOLO il frontend React.

──────────────────────────────

CHAT

1. Le bolle devono avere:
- margini corretti
- larghezza massima 70%
- angoli arrotondati stile Signal
- ombra leggera
- padding maggiore
- timestamp più piccolo
- stato del messaggio (inviato/consegnato/letto) predisposto.

2. Animazioni:
- comparsa messaggi
- scroll fluido
- auto-scroll solo se l'utente è già in fondo
- transizioni morbide.

──────────────────────────────

INPUT

Ridisegnare completamente la barra inferiore.

Da sinistra verso destra:

😊 Emoji
📎 Allega
Campo testo espandibile
🎤 Microfono
➡️ Invia

Il pulsante invio compare solo se il campo contiene testo.
Altrimenti mostra il microfono.

Il campo cresce automaticamente fino a 6 righe.
